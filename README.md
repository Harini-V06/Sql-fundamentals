# SQL Basics Project

This repository demonstrates fundamental SQL concepts using MySQL.

## Using this Repository
- Create a database in MySQL
- Run table creation scripts
- Insert sample data using insert queries
- Execute select queries to practice joins and aggregate functions.


## Topics Covered
- Database and table creation
- Primary and foreign keys
- CRUD operations
- Joins
- Aggregate functions
